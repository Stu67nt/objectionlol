# objectionlol

This is a visual novel game (debatable) where you get to watch and participate in the most brain cell-destroying arguments you will ever hear. 

This project is for Hack Club's #ResolutionGameDev event, where each week you work on a game dev project relating to a theme! This project is week 1 of Resolution, where I needed to create the basis of a visual novel. For this, I decided to recreate a YouTube video argument I watched [linked here](https://www.youtube.com/watch?v=4DR0-xzw1NE).

For week 1 I have just the basic text and character sprites, unanimated. I also have a kinda useless player choice which I was required to put in for the purposes of passing the week. 

For week 2 I added all the music and sfx as well as a way to remember whether the user has checked each ending.
